Estrategia de stock offline
Al vender offline → el stock se decrementa localmente en Dexie de forma optimista.
Al reconectar → la venta se sube a Supabase y el stock real se reconcilia.
Si hubo discrepancia (otro dispositivo vendió el mismo producto) → Supabase gana. El stock local se refresca con el valor correcto.
